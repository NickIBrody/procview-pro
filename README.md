# procview-pro
ProcView Pro — a Windows process monitor with real-time search. Shows memory/CPU usage, has pause/resume and fast/normal update modes. Portable EXE — no Python installation required.

ProcView Pro — Windows-утилита для мониторинга процессов с поиском в реальном времени. Показывает использование памяти/CPU, имеет режимы паузы и быстрого обновления. Portable EXE — не требует установки Python.

## System Requirements
- **OS:** Windows 7/8/10/11 (64-bit or 32-bit)
- **RAM:** 512 MB minimum
- **Disk space:** 10 MB
- **No Python required** for the EXE version
